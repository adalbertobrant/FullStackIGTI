Diferenças entre var, const, let

O javascript moderno tende a não utilizar mais variáveis do tipo global, como são as inicializadas por var em vez disso é preferível inicializar a variável por const ou let caso tenha um escopo mais local.

Um fato interessante é que variáveis do tipo const podem ter o seu conteúdo alterado caso façam parte de um objeto ou array.

Arrow Functions

Arrow function é uma nova forma de construir funções de uma maneira mais simplificada e intuitiva em javascript, aumentando a simplicidade e diminuindo a quantidade de palavras.

Template Literals

Os template literals são uma nova forma de concatenar e escrever strings dando mais agilidade ao código e rapidez.

Default Parameters

É a possibilidade de deixar um valor por default na função caso a mesma venha sem um paramêtro expresso e também serve para verificar se a função com parametro é válida.
