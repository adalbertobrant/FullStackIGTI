Rest / Spread Operadores e Destructuring

Operador spread ( ... )

Dado um array com seis posições let array = [1,2,3,4,5,6], para realizar a iteração sobre o mesmo, é necessário uma função. A função spread "abre" o array, literalmente espalhando o mesmo e possibilitando sua iteração, "..." é útil em situação para iteração de arrays, strings, objetos e qualquer outra necessidade de iteração.

Operador rest ( ... )

O operador rest (...) é usado para agrupar elementos para passagem de parametros para uma função. Ele pode ser utilizado ao final do argumento de uma função.

Operador destructuring

Faz atribuição via desestruturação de um obejto complexo, tendo as chaves daquele objeto já definidas pelo usuário. Também pode ser utilizada para matrizes.
