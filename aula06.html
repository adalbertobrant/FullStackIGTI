Manipulação no DOM

O DOM ou modelo de objetos de documento ou em inglês Document Object Model, se refere a uma maneira de organizar toda uma página web em uma estrutura de árvore com pais e filhos, para que se possa encontrar partes do html de maneira mais rápida e simples. A manipulação do DOM normalmente é feita utilizando frameworks pois tem um custo computacional. Listamos a seguir os comandos e breve explicação.

    querySelector()
- Pode acessar elementos no html um a um sendo instanciado sempre a sua primeira ocorrência.

var paragrafo = documento.querySelector('p');

    querySelectorAll()
- Retorna elementos do tipo node list ou seja uma lista linkada com os pais e filhos. Pode ser feita uma conversão de nós para Array da seguinte forma:

    var x = Array.from(document.querySelectorAll('p'));

    textContent
- Retorna o conteúdo textual de alguns elementos.

elemento.textContent = "Mudando elemento";
