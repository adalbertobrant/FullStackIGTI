A manipulação de eventos em Javascript, utiliza o evento como uma cadeia temporal do tipo QUANDO. É necessário então que o código em javascript sempre fique "escutando" alterações nos inputs de um formulário ou outra ação que se fizer necessária, como a rolagem da tela, passar do mouse, etc. O javascript utiliza a seguinte função addEventListener para procurar alterações durante a execução do html.
